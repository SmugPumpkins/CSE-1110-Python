# Lesson 01 - Hello World

## Overview

This lesson introduces your first Python program and demonstrates how to display output in the console using the `print()` function.

## `print()`

The `print()` function displays text or other data in the console so the user can see it.

* `print()`: A built-in Python function used to display output.
* `"Hello World"`: A string (text) enclosed in quotation marks.
* Parentheses `()`: Required to call (run) the function.
* Quotes `""`: Indicate that the content inside is text.

```python
# Display a message in the console
print("Hello World")
```
